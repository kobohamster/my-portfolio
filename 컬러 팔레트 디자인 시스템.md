## 컬러 사용 가이드

### Header / Navigation

```css
header {
  background-color: var(--color-bg-primary);    /* #202020 */
  color: var(--color-text-primary);             /* #FFFFFF */
  border-bottom: 1px solid var(--color-border); /* #603040 */
}
```

- 로고·네비 링크: `--color-text-primary` (`#FFFFFF`)
- 활성화된 메뉴: `--color-accent` (`#F0A0C0`) + 언더라인
- 스크롤 시 배경: `--color-bg-primary`에 `opacity: 0.95` 적용

## 반응형 고려사항

### 다크 모드 (prefers-color-scheme)

```css
/* 라이트 모드 기본값 */
:root {
  --color-bg-primary:    #FFB0D0;
  --color-bg-secondary:  #FFFFFF;
  --color-text-primary:  #302030;
  --color-text-muted:    #6060B0;
}

/* 다크 모드 오버라이드 */
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary:    #202020;
    --color-bg-secondary:  #302030;
    --color-text-primary:  #FFFFFF;
    --color-text-secondary: #FFB0D0;
    --color-text-muted:    #E090B0;
    --color-border:        #603040;
    --color-accent:        #F0A0C0;
  }
}
```

### 모바일 접근성 (contrast ratio)

| 조합 | 비율 | WCAG 등급 |
|------|------|-----------|
| `#FFFFFF` on `#202020` | ~16:1 | AAA ✅ |
| `#FFFFFF` on `#6060B0` | ~4.6:1 | AA ✅ |
| `#302030` on `#FFB0D0` | ~8.1:1 | AAA ✅ |
| `#E090B0` on `#202020` | ~5.2:1 | AA ✅ |

> 모든 주요 텍스트 조합이 WCAG 2.1 AA 기준(4.5:1) 이상을 만족합니다.

## MUI 테마 적용 (포트폴리오 프로젝트용)

```js
// src/theme.js
import { createTheme } from '@mui/material/styles'

const theme = createTheme({
  palette: {
    mode: 'dark',
    primary: {
      main:  '#6060B0',   // 페리윙클 퍼플
      light: '#7070B0',   // 미디엄 퍼플블루
      dark:  '#5060A0',   // 딥 블루퍼플
    },
    secondary: {
      main:  '#F0A0C0',   // 로즈 핑크
      light: '#FFB0D0',   // 블러시 핑크
      dark:  '#D080A0',   // 더스티 로즈
    },
    background: {
      default: '#202020', // 다크 플럼
      paper:   '#302030', // 다크 로즈그레이
    },
    text: {
      primary:   '#FFFFFF',  // 기본 텍스트
      secondary: '#FFB0D0',  // 서브 텍스트
      disabled:  '#E090B0',  // 비활성 텍스트
    },
    divider: '#603040',       // 다크 로즈 테두리
  },
})

export default theme
```
