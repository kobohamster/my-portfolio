## CSS 변수 정의

```css
:root {
  /* ── Primary Colors ── */
  --color-primary:       #4A46AA;   /* 메인 포인트 컬러 (중간 퍼플) */
  --color-primary-light: #6B5CC5;   /* 밝은 버전 — 호버, 하이라이트 */
  --color-primary-dark:  #1E1B6B;   /* 어두운 버전 — 헤더, 히어로 섹션 */

  /* ── Secondary Colors ── */
  --color-secondary:     #6B5CC5;   /* 보조 포인트 컬러 */
  --color-accent:        #EDE8F8;   /* 강조 패널·라이트 배경 */

  /* ── Background Colors ── */
  --color-bg-primary:    #1E1B6B;   /* 메인 페이지 배경 (다크) */
  --color-bg-secondary:  #EDE8F8;   /* 섹션 교체 배경 (라이트) */

  /* ── Text Colors ── */
  --color-text-primary:  #FFFFFF;   /* 다크 배경 위 기본 텍스트 */
  --color-text-secondary: #2A246E;  /* 라이트 배경 위 텍스트 */
  --color-text-muted:    #9B93D9;   /* 캡션·서브텍스트·비활성 */

  /* ── Interactive Colors ── */
  --color-button-primary: #6B5CC5;  /* 기본 버튼 */
  --color-button-hover:   #1E1B6B;  /* 버튼 호버 */
  --color-link:           #9B93D9;  /* 링크 기본 */
  --color-link-hover:     #FFFFFF;  /* 링크 호버 */

  /* ── Border & Divider ── */
  --color-border:        #3D38A0;   /* 테두리·구분선 */
  --color-border-light:  #C5BFEA;   /* 라이트 섹션 내 테두리 */

  /* ── Shadow ── */
  --shadow-card: 0 4px 24px rgba(30, 27, 107, 0.25);
  --shadow-button: 0 2px 8px rgba(107, 92, 197, 0.4);
}
```

## 컬러 사용 가이드

### Header / Navigation

```css
header {
  background-color: var(--color-primary-dark);  /* #1E1B6B */
  color: var(--color-text-primary);             /* #FFFFFF */
  border-bottom: 1px solid var(--color-border); /* #3D38A0 */
}
```

- 로고·네비 링크: `--color-text-primary` (`#FFFFFF`)
- 활성화된 메뉴: `--color-secondary` (`#6B5CC5`) + 언더라인
- 스크롤 시 배경: `--color-primary-dark`에 `opacity: 0.95` 적용

---

### Hero Section

```css
.hero {
  background: linear-gradient(
    135deg,
    var(--color-primary-dark) 0%,   /* #1E1B6B */
    var(--color-primary) 100%       /* #4A46AA */
  );
}
.hero h1 { color: var(--color-text-primary); }   /* #FFFFFF */
.hero p  { color: var(--color-text-muted); }     /* #9B93D9 */
```

### Buttons

```css
/* Primary Button */
.btn-primary {
  background-color: var(--color-button-primary);  /* #6B5CC5 */
  color: var(--color-text-primary);               /* #FFFFFF */
  border: none;
  box-shadow: var(--shadow-button);
}
.btn-primary:hover {
  background-color: var(--color-button-hover);    /* #1E1B6B */
}

/* Outline Button */
.btn-outline {
  background-color: transparent;
  color: var(--color-secondary);                  /* #6B5CC5 */
  border: 2px solid var(--color-secondary);
}
.btn-outline:hover {
  background-color: var(--color-secondary);
  color: var(--color-text-primary);
}
```

## 반응형 고려사항

### 다크 모드 (prefers-color-scheme)

```css
/* 라이트 모드 기본값 (라이트 배경 섹션용) */
:root {
  --color-bg-primary:   #EDE8F8;
  --color-bg-secondary: #FFFFFF;
  --color-text-primary: #2A246E;
  --color-text-muted:   #6B5CC5;
}

/* 다크 모드 오버라이드 */
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary:    #1E1B6B;
    --color-bg-secondary:  #0D0B3D;
    --color-text-primary:  #FFFFFF;
    --color-text-muted:    #9B93D9;
    --color-border:        #3D38A0;
    --color-accent:        #2D2878;
  }
}
```

### 모바일 접근성 (contrast ratio)

| 조합 | 비율 | WCAG 등급 |
|------|------|-----------|
| `#FFFFFF` on `#1E1B6B` | ~11:1 | AAA ✅ |
| `#FFFFFF` on `#4A46AA` | ~5.8:1 | AA ✅ |
| `#2A246E` on `#EDE8F8` | ~9.2:1 | AAA ✅ |
| `#9B93D9` on `#1E1B6B` | ~4.6:1 | AA ✅ |

> 모든 주요 텍스트 조합이 WCAG 2.1 AA 기준(4.5:1) 이상을 만족합니다.

### 화면 크기별 색상 전략

```css
/* 모바일: 그라디언트 대신 단색으로 성능 최적화 */
@media (max-width: 768px) {
  .hero {
    background: var(--color-primary-dark);  /* 그라디언트 제거 */
  }
}
```

## MUI 테마 적용 (포트폴리오 프로젝트용)

```js
// src/theme.js
import { createTheme } from '@mui/material/styles'

const theme = createTheme({
  palette: {
    mode: 'dark',
    primary: {
      main:  '#6B5CC5',   // --color-secondary
      light: '#9B93D9',   // --color-text-muted
      dark:  '#1E1B6B',   // --color-primary-dark
    },
    secondary: {
      main: '#EDE8F8',    // --color-accent
    },
    background: {
      default: '#1E1B6B', // --color-bg-primary
      paper:   '#4A46AA', // --color-primary
    },
    text: {
      primary:   '#FFFFFF', // --color-text-primary
      secondary: '#9B93D9', // --color-text-muted
    },
    divider: '#3D38A0',     // --color-border
  },
})

export default theme
```
